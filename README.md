# EcoTrack — Community Sustainability Platform



Features:
- Discover and join measurable community sustainability challenges.
- Track personal progress and view community totals (CO₂ saved, plastic reduced).
- Share and upvote practical eco-tips and browse local green events.
- Firebase Authentication (Email/Password + Google) and secure server APIs.
- Responsive UI with skeleton loaders, toast notifications, and accessible markup.

Tech stack:
- React, Tailwind CSS, Firebase Auth, react-router v6
- Server: Node.js, Express, MongoDB (deployed separately)

How to run locally:
1. `cd client` → `npm install`
2. create `.env.local` with VITE_* variables and API base URL
3. `npm run dev`

Contributing / Git commits:
- Follow the commit plan in the project spec; ensure at least 15 client commits and 8 server commits.
